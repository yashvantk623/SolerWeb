## 📥 IMPLEMENTATION GUIDE

### Where to Add Images in Blog System

**Update `/app/blog/blogs.json` to include image fields:**

```json
{
  "id": 1,
  "slug": "Haryana & Utta Pradesh-solar-roi-4-years",
  "title": "...",
  "featuredImage": "/blog-images/blog-1-hero.jpg",
  "inlineImages": [
    {
      "name": "ROI Breakdown",
      "src": "/blog-images/blog-1-roi-breakdown.jpg",
      "caption": "Annual savings breakdown showing 4-year payback"
    }
  ],
  ...
}
```

### Update Blog Detail Page (`/app/blog/[slug]/page.tsx`)

```tsx
// Add featured image at top
<img 
  src={blog.featuredImage} 
  alt={blog.title}
  className="w-full h-auto rounded-lg mb-8 shadow-lg"
/>

// Add inline images throughout content
{blog.inlineImages?.map((img) => (
  <figure key={img.name} className="my-8">
    <img 
      src={img.src} 
      alt={img.name}
      className="w-full rounded-lg shadow-md"
    />
    <figcaption className="text-sm text-gray-600 text-center mt-2">
      {img.caption}
    </figcaption>
  </figure>
))}
```
